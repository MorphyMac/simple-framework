# Simple Framework
This library includes useful tools for the IST 311 course at Penn State University. Tools include categories like:
* ###concurrent
    * `IObserver`
    * `OnTaskStateListener`
    * `Task`

* ###framework
    * `Bundle`
    * `AppDialog`
    * `AppFrame`
    * `IFrameManager`
    * `FrameManager`

* ###security
    * `ICryptography`
    * `Base64`
    * `MD5DES`
    * `AES`
    * `Hasher`

* ###sql
    * `ContentValues`
    * `SQLBuilder`
    * `SQLiteCloseable`
    * `SQLiteDatabase`
    * `SQLiteOpenHelper`
    * `SQLiteOpenHelper2`

* ###storage
    * `ISharedPreferences`
    * `SharedPreferences`
    * `PreferenceManager`

* `Log`
* `TextUtils`
